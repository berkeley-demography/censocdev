Summary: This vignette gives an overview of the CenSoc-DMF Demo dataset and presents two stylized examples of how it can be used to answer a research question.

The CenSoc-DMF Demo dataset was constructed by (i) linking the CenSoc-DMF dataset to the IPUMS 1940 1% census sample and (ii) selecting a set of 20 mortality covariates from the 1940 census. The smaller size of the file — approximately 1% of the records in the full CenSoc-DMF dataset — makes it easier to work with but precludes the high-resolution mortality research possible with the full CenSoc-DMF dataset.

Examples with OLS Regression

Using OLS Regression on age of death with fixed effect terms for each year of birth is a straightforward method for analyzing CenSoc mortality data. There are a few specific considerations of using regression on age of death to analyze the CenSoc Mortality data. We are only observing deaths for the window of 1975-2005 in the CenSoc-DMF file. As the left and right truncation ages vary by birth cohort, it is important to include fixed effect terms for each year of birth. Models of the form:

\[ Age\_at\_death = birth\_year\_dummy + covariates\_of\_interest \]

provide estimates of the effect of the covariates on the age of death in the sample, controlling for birth cohort truncation effects. In this example, we work with the cohorts of 1900-1920, so we need to include a fixed effect term for year of birth.

As the truncated window of deaths excludes the tails of the mortality distribution, any measurement of the average difference between groups will be downwardly biased. When we interpret the coefficients in our OLS regression model, we need to keep in mind that the observed effects likely underestimate the true effects.

Mortality Differentials by State

The code below runs an OLS model with year fixed-effects. It then extracts the regression coefficient for each state and plots it using the statebins package (an alternative to state-level choropleth maps). We use the state of Maine as the reference group for this regression (due to its average effect on mortality).

## Prepare dataset for modeling
## Restrict to deaths for persons age 65+
## Restrict to birth cohorts of 1900-1920

censoc_state_model <- censoc_demo %>%
  filter(byear %in% c(1900:1920)) %>% 
  mutate(byear = as.factor(byear)) %>% 
  mutate(byear = relevel(byear, ref = "1910")) %>% 
  mutate(STATEFIP = as.factor(STATEFIP)) %>% 
  mutate(STATEFIP = relevel(STATEFIP, ref = "Maine")) 

## Linear model predicting age at death from State and byear 
state.lm <- lm(death_age ~ STATEFIP +  byear,
                     data = censoc_state_model,
                     weight = weight*PERWT) 

## Put model results into a data.frame 
state.lm.df <- tidy(state.lm)

## Select coefficients and ZIP Codes
state.lm.df <- state.lm.df %>%
  select(term, estimate) %>% 
  filter(str_detect(term, "STATEFIP")) %>% 
  mutate(state = substr(term, 9, 35)) %>% 
  select(state = state, value = estimate)

## Plot using State Bins Package
mortality_differentials_by_state <- statebins_continuous(state_data = state.lm.df, state_col = "state",
                      value_col ="value", font_size = 5, brewer_pal = "Blues",
                     legend_title = "Mortality Differentials",
                     legend_position = "bottom") + 
  labs(title = "Mortality Differentials by State") + 
  theme(plot.title = element_text(size=22))

## Display Plot
mortality_differentials_by_state

The plot shows the Mountain States have a relative mortality advantage and the Southern States to have a relative mortality disadvantage. While these patterns may be real, we should keep in mind disaggregating by state significantly reduces our sample size. This introduces lots of noise and uncertainty into our regression model, which our visualization doesn’t account for.

Mortality Differentials by Wage Income

The association between relative income and mortality in the United States have been well documented. Can we see this association in our CenSoc-DMF Demo dataset?

To answer this question, we use the INCWAGE variable. This is an imperfect measure of income, as it was only collected for salary and wage workers. Census enumerators in 1940 were explicitly instructed to “not include earnings of businessmen, farmers, or professional persons who depend upon business profits, sales of crops, or fees for income and who do not work for wages or salaries.” (See instructions to enumeration for more detail). In this analysis, we account for this by only including persons with non-zero wages — other researchers may take a different approach.

## Calculate income deciles
censoc_incwage <- censoc_demo %>% 
  filter(INCWAGE > 0 & INCWAGE <= 5001) %>% ## INCWAGE is topcoded at $5001; higher values denote NA/Missing
  mutate(wage_decile = ntile(INCWAGE, 10)) %>% 
  mutate(wage_decile = as.factor(wage_decile))

## Run linear model 
test <- lm(death_age ~  wage_decile +  byear + as.factor(RACE),
                     data = censoc_incwage,
                     weight = weight*PERWT) 

## Put model results into a data.frame 
test.df <- tidy(test, conf.int = T)
  
## Select coefficients and standard errors from OLS model
test.df <- test.df %>%
  select(term, estimate, se = std.error) %>% 
  filter(str_detect(term, "wage_decile")) %>% 
  mutate(quantile = as.numeric(substr(term, 12, 15))) %>% 
  select(quantile, value = estimate, se)

## Plot 
income_decile_ols_coefficients <- ggplot(data = test.df) + 
  geom_pointrange(aes(ymin = value - se, ymax = value + se, y = value, x = quantile)) + 
  theme_bw(base_size = 15) + 
  labs(title = "Income pattern of Longevity at Age 65",
       x = "Wage Income Decile",
       y = "Additional Years of Life")

## Display Plot
income_decile_ols_coefficients

The plot shows an association between relative wage income and longevity (the reference group for this plot is the first income decile). When interpreting these mortality differentials, we should keep in mind that they probably understate the true effect.

Conclusion

The CenSoc-Demo dataset allows researchers to quickly explore broad patterns of mortality differentials in the United States. However, it isn’t conducive to high-resolution mortality research, and we recommend working with the complete CenSoc-DMF file for any final analysis.
